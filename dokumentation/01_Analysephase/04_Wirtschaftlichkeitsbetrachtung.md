# Wirtschaftlichkeitsbetrachtung (1 Stunde)

* Ergebnis: Kosten-Nutzen-Analyse
    
* Kostenentwicklung:
    | Kostenart       | Externe Lösung (z.B. Guestboard)   | Eigener Prototyp |
    | ------------- |:------------------:| -----:|
    | Einmalkosten | 0€   | Entwicklungskosten (20 Stunden) |
    | Monatliche Kosten | ~100-200€/Monat |   ~10€/Monat (Firebase) |
    | Wartung/Support | Inklusive         | Eigener Aufwand |
    | Jahreskosten (1. Jahr) | 1.200-2.400€         | ~120€ + Entwicklung |

* Entwicklungskosten (theoretisch):
    ```
    20 Stunden × 50€/Stunde (Junior Entwickler) = 1.000€ 
    Für Ausbildungskontext: 0€ (Lerninvestition)
    ```

* Einsparpotenziale:
    * Zeiteinsparung Rezeption:
        * ```Aktuell: ~2 Stunden/Tag für Service-Anfragen telefonisch```
        * ```Mit PWA: ~0,5 Stunden/Tag für digitale Bearbeitung```  
        * ```Einsparung: 1,5 Stunden/Tag × 20€/ Stunden × 365 Tage = 10.950€/Jahr```  
    * Erhöhte Gästezufriedenheit:
        * ```Bessere Bewertungen → Mehr Buchungen```
        * ```Konservativ geschätzt: +2% Auslastung```
        * ```80 Zimmer × 150€/Nacht × 2% × 365 Tage = 87.600€ zusätzlicher Umsatz/Jahr```
    * Reduzierter Papierverbrauch:
        * ```Aktuell: ~50 Service-Zettel/Tag```
        * ```Einsparung: 50 × 0,05€ × 365 = 912€/Jahr```
