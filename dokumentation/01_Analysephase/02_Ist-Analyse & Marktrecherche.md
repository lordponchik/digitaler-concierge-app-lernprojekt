# Ist-Analyse & Marktrecherche (1 Stunde)

* Ergebnis: Prozessanalyse und Marktvergleich
    
* Aktuelle Prozesse im Hotel:

    | Prozess       | Aktueller Ablauf   | Probleme |
    | ------------- |:------------------:| -----:|
    | Service-Anfrage | Telefonanruf → Rezeption → Papierzettel → Weiterleitung    | Wartezeiten, Missverständnisse, Papierchaos |
    | Info-Bereitstellung | Gedruckte Flyer, veraltete Website |   Keine Aktualität, schlechte mobile Darstellung |
    | Check-in | Persönlich an Rezeption mit Wartezeit         | Schlangenbildung, Personalbindung |

* Marktrecherche - Vergleichbare Lösungen:

    * "Mews" - Vollständiges PMS (zu komplex für Prototyp)

    * "Alice" - Hotel-Kommunikationsplatform (Enterprise-Lösung)

    * "Guestboard" - Digitale Concierge-App (kostenpflichtig, monatliche Gebühren)

> __Erkenntnis:__ Es gibt professionelle Lösungen ab 100€/Monat. Unser Prototyp kann zeigen, dass eine individuelle Lösung kostengünstiger möglich ist.

* Technische Infrastruktur Hotel:

    * WLAN im gesamten Hotel
    * 3 Rezeptions-PCs (Windows 10)
    * Keine API-Schnittstellen vorhanden
    * Keine IT-Fachkraft vor Ort
